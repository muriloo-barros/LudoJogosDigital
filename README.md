## 📖 Sobre o Projeto

O **LudoGames** é uma plataforma web de jogos educacionais desenvolvida com foco em **acessibilidade e inclusão**. O projeto oferece jogos voltados para crianças de 6 a 8 anos, abordando conteúdos como matemática, leitura, ciências, lógica e habilidades socioemocionais de forma leve e descontraída.

A plataforma foi pensada para atender crianças com diferentes necessidades — **auditivas, visuais, motoras e cognitivas** — garantindo que todos possam aprender de maneira fácil e confortável.

---

## ✨ Funcionalidades

- 🔊 **Narração por voz** — leitura automática de textos ao passar o mouse ou clicar, com fila de áudio para evitar sobreposição
- 📝 **Legendas animadas** — exibição visual do texto sendo narrado em tempo real
- 🤟 **VLibras** — integração com o plugin oficial de Língua Brasileira de Sinais
- 🖱️ **Cursor personalizado** — experiência visual imersiva e lúdica
- 🎨 **Gradiente dinâmico no header** — transição de cores conforme o scroll da página
- ♿ **Acessibilidade** — uso de `aria-hidden`, `aria-live`, `alt` descritivo e semântica HTML correta

## 🚀 Como Rodar Localmente

Não é necessário instalar nada — basta clonar e abrir no navegador.

```bash
# Clone o repositório
git clone https://github.com/Mythos1233/LudoGamesDigital.git

# Entre na pasta
cd LudoGamesDigital

# Abra o arquivo no navegador
# Clique duas vezes em index.html, ou use uma extensão como Live Server (VS Code)
```

> **Recomendado:** usar a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) no VS Code para evitar problemas com caminhos de arquivo locais.
